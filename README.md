# ChocoLog — チョコザップ広島八木店 トレーニング記録アプリ

スマホのホーム画面に追加して使えるPWA（Progressive Web App）です。

## ファイル構成

```
chocozap-tracker/
├── index.html        # アプリ本体（全機能入り）
├── manifest.json     # PWA設定
├── sw.js             # Service Worker（オフライン対応）
├── icons/
│   ├── icon-192.png  # アプリアイコン
│   └── icon-512.png  # アプリアイコン（大）
└── README.md
```

## GitHub Pagesで公開する手順

1. GitHubで新しいリポジトリを作成（例: `chocozap-tracker`）
2. このフォルダのファイルをすべてアップロード
3. Settings → Pages → Branch: `main` / root → Save
4. `https://あなたのユーザー名.github.io/chocozap-tracker/` でアクセス可能に

## スマホのホーム画面に追加する方法

### iPhone（Safari）
1. Safariで上記URLを開く
2. 下部の共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択

### Android（Chrome）
1. Chromeで上記URLを開く
2. メニュー（⋮）→「ホーム画面に追加」

## データについて

記録データはブラウザの **localStorage** に保存されます。

- 同じブラウザ・同じURLであればデータは保持されます
- ブラウザのデータを消去（キャッシュクリア）すると消えます
- データのバックアップは **CSV出力** タブからできます
- 機種変更時はCSVを保存しておくことをおすすめします
